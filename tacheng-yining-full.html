0 4/30（四）✈️ 成都→塔城落地 · 提车日
~10km · 半天 · 适应环境
  • 09:00 成都起飞 → 约13:30 塔城落地（飞行约4.5h）
  • 14:30 塔城机场提车（租车周期：4/30 14:30 → 5/6 13:00 = 6天整）
  • 开车进塔城市区 → 办理入住 → **傍晚闲逛**塔城市区
  • 推荐：双河公园(市区散步) / 手抓羊肉(当地特色) / 塔城俄式建筑街拍
  • 💡 塔城是俄罗斯族聚居区，城市氛围悠闲，适合落地第一天放松适应
  • 🏨 住宿：塔城市区（今晚好好休息，明天开始玩！）
  • 💰 预估：油费¥10 + 餐饮¥50 + 住宿¥350 = ~¥410
1 5/1（五·劳动节）塔城周边慢游日 ☀️
~90km往返 · 不赶路 · 五一但不堵
  • 睡个自然醒 → 早餐吃塔城面包/列巴（俄式风味）
  • 上午：巴克图口岸 🚩 — 中哈边境国门！远眺哈萨克斯坦，边境打卡圣地
  • 中午驱车前往裕民县（S221省道，约45km/1h）
  • 下午：小白杨哨所 🎵 — 《小白杨》歌曲发源地！边防精神象征，必打卡
  • 裕民县逛逛 → 尝尝当地美食 → **返回塔城**（或直接住裕民也行）
  • 💡 五一当天但在塔城地区，游客比热门景区少很多，体验好！
  • 🏨 住宿：塔城市区（连续第二晚，不换房！）或裕民县
  • 💰 预估：油费¥40 + 餐饮¥80 + 住宿¥350 = ~¥470
2 5/2（六）🚗 正式向西出发！塔城→精河县
~280km · ~4h ⚠️ 最长驾驶日
  • ⚠️ 今天开始长途驾驶，建议 **08:00 出发**
  • 上午：塔城/裕民 → 巴尔鲁克山风景区 — 高山草原、吐尔加辽丘陵草原、野生玫瑰谷🌹（五月初玫瑰初开！）
  • 中午经S221→G3015转G30前往阿拉山口口岸
  • 阿拉山口 🚧 — 中哈边境国门，著名的"大风口"。远眺哈萨克斯坦
  • 下午G30高速到精河县 — 精河凉皮必吃！！（¥12-18/碗）
  • 💡 五一假期已过最高峰，高速路况应该比5/1好很多
  • 🏨 住宿：精河县（今晚好好休息，明天是重头戏！）
  • 💰 预估：油费¥140 + 过路费¥50 + 门票¥80(巴尔鲁克) + 餐饮¥100 + 住宿¥450 = ~¥820
3 5/3（日）精河→赛里木湖💎→果子沟🌉→抵达伊宁！
~250km · 全天亮点🌟
  • 早上走G30高速 → 赛里木湖（约101km / 1.5h）
  • 💎 赛里木湖 — "大西洋最后一滴眼泪" 今日绝对主角！！
  • 推荐玩法：东门进 → 克勒涌珠 → 松树头观台 → 月亮湾 → 南门出（预留3-4小时）
  • 门票¥70 + 区间车¥75（或自驾环湖公路只买门票¥70）
  • 果子沟大桥 🌉 — 亚洲最大公路隧道群！多个免费观景台拍照绝美
  • **下午继续沿G30直奔伊宁市**（不再住清水河了！）
  • 傍晚抵达伊宁 → 办理入住 → 晚上逛喀赞其民俗村（蓝房子街区！手工冰淇淋¥5-8！）
  • 🏨 住宿：伊宁市（接下来3晚都不用换了！🎉）
  • 💰 预估：赛湖门票¥145 + 油费¥70 + 过路费¥30 + 餐饮¥120 + 住宿¥500 = ~¥865
4 5/4（一）伊宁↔霍尔果斯🛍️+图开沙漠🏜️ 往返
~150km往返 · 轻松辐射 · 不换酒店
  • 以伊宁为基地的一日辐射游，早出晚归，行李丢车里不动 😎
  • 上午：伊宁出发G30 → 霍尔果斯口岸（~65km / 1h）
  • 🛍️ 中哈合作中心免税购物 — 哈国巧克力/蜂蜜/零食，记得带身份证！
  • 第六代国界碑打卡 → 中哈两国风情体验
  • 中午：图开沙漠 🏜️ — 新疆唯一绿洲中的沙漠！奇特景观反差（~25km）
  • 下午**返回伊宁** → 傍晚逛六星街（俄罗斯风情街/手风琴博物馆）
  • 晚上：丝路之光夜市 或 汉人街大巴扎（手抓饭¥35-48 / 卡瓦斯¥6-10）
  • 🏨 住宿：伊宁市（连续第二晚）
  • 💰 预估：油费¥65 + 餐饮¥130 + 住宿¥500 = ~¥695
5 5/5（二）伊宁环线游 · 托乎拉苏⛰️+天山花海💐
~130km环线 · 或改市内游
  • 方案A（自然风光环线）：以伊宁为基地的一日环线
  • 上午：托乎拉苏草原 ⛰️（距伊宁~50km / 1h）— 空中草原、河谷风光、哈萨克毡房
  • ⚠️ 5月初草原刚返青，草色嫩黄嫩绿的也别有一番风味
  • 下午：天山花海 💐（霍城县芦草沟镇）— 万亩薰衣草基地
  • ⚠️ 花期预警：薰衣草通常6月中旬盛开！5月初只能看到绿油油的幼苗田
  • 方案B（推荐·伊宁市内深度游）：如果不想看"绿色的麦田"
  • 上午：喀赞其二刷 — 上次没拍够的照片补拍！马车巡游深度体验
  • 中午：林则徐纪念馆 → 伊犁河风景区（日落超美！）
  • 下午：汉人街买特产 — 干果/奶酪/馕/手工皂，伴手礼采购
  • 傍晚：伊犁河大桥日落 → 回酒店收拾行李准备明天的返程
  • 🏨 住宿：伊宁市（连续第三晚，最后一晚！）
  • 💰 预估(A)：油费¥75 + 门票¥80(托乎拉苏) + 餐饮¥130 + 住宿¥500 = ~¥785
  • 💰 预估(B)：油费¥20 + 餐饮¥150 + 住宿¥500 = ~¥670（市内游更省钱）
6 5/6（三）伊宁收尾 → ✈️ 飞回成都
~5km · 上午半天 · 返程日
  • 最后半天！睡个懒觉 → 退房
  • 上午最后逛：六星街早餐（俄罗斯面包+咖啡）→ 再去喀赞其买个冰淇淋告别
  • 或者去苹果便利店补点路上零食（飞机餐不好吃😄）
  • ⚠️ 13:00 伊宁机场还车（留足检查+加油时间！别迟到！）
  • 16:00 伊宁起飞 → 约20:00 成都落地 → 回家！
  • 🏨 无需住宿（当晚回到温暖的家）
  • 💰 预估：餐饮¥60 = ~¥60（不含特产购物费用）
