📍 每日详细行程（航班+自驾版）
1 4/30（四）✈️ 塔城落地 → 裕民县 · 小白杨哨所
69 + 76 = ~45km · 半天 · 提车日
  • 09:00 成都起飞 → 约13:30-14:00 塔城落地（飞行约4.5h）
  • 14:30 塔城机场提车（租车周期：4/30 14:30 → 5/6 13:00 = 6天整）
  • S221省道向南 → 裕民县（~45km / ~1h车程）
  • 小白杨哨所 🎵 — 歌曲《小白杨》发源地！边防精神象征，打卡必去
  • 裕民县晚餐 + 适应环境
  • 🏨 住宿：裕民县
  • 💰 预估：油费¥35 + 餐饮¥60 + 住宿¥200 = ~¥295
2 5/1 托里 → 精河县
232km · ~3.5h
  • ⚠️ 五一黄金周第一天，建议8:00前出发避开人流高峰
  • 上午：巴尔鲁克山风景区 — 高山草原、吐尔加辽丘陵草原、野生玫瑰谷🌹
  • 中午经S221→G3015转G30前往阿拉山口口岸
  • 阿拉山口 🚧 — 中哈边境国门，远眺哈萨克斯坦。⚠️ 五一可能排队，预留1-2h
  • 下午G30到精河县 — 精河凉皮必吃！！（¥12-18/碗）
  • 🏨 住宿：精河县
  • 💰 预估：油费¥140 + 过路费¥50 + 门票¥80(巴尔鲁克) + 餐饮¥100 + 住宿¥450 = ~¥820
3 5/2 精河 → 赛里木湖（全天亮点🌟）
101km · ~1.5h
  • 早上走G30高速 → 赛里木湖（~101km / ~1.5h）
  • 💎 赛里木湖 — "大西洋最后一滴眼泪" 核心亮点！
  • 环湖推荐：东门进 → 克勒涌珠 → 松树头观台 → 南门出
  • 门票¥70 + 区间车¥75（或只买门票¥70环湖公路自驾）
  • 果子沟大桥 🌉 — 亚洲最大公路隧道群！多个免费观景台
  • 下山住清水河镇（距赛湖约30min）
  • 🏨 住宿：清水河镇（离霍尔果斯近方便明日）
  • 💰 预估：门票¥145 + 油费¥70 + 餐饮¥120 + 住宿¥480 = ~¥815
4 5/3 果子沟 → 特克斯八卦城
342km · ~4.5h ⚠️ 最长一天
  • 上午：霍尔果斯口岸 — 中哈合作中心（免税购物）、第六代国界碑
  • 🛍️ 免税区可买哈国巧克力、蜂蜜等，记得带身份证
  • 中午：图开沙漠 🏜️ — 新疆唯一绿洲中的沙漠！奇特景观反差
  • 下午抵达伊宁市 — 进入伊犁河谷核心区！
  • 喀赞其民俗旅游区 🏠 — 蓝房子街区！手工冰淇淋（¥5-8）、马车巡游、维吾尔风情
  • 晚上逛伊宁：汉人街大巴扎、手抓饭（¥38）、卡瓦斯
  • 🏨 住宿：伊宁市（D5也住这不用换！）
  • 💰 预估：油费¥65 + 餐饮¥130 + 住宿¥500 = ~¥695
5 5/4 特克斯 → 库尔德宁 → 新源
153km · ~2.5h
  • 以伊宁为基地的一日环线，早出晚归不换酒店
  • 上午：托乎拉苏草原 ⛰️（距伊宁~50km / ~1h）— 空中草原、河谷风光、哈萨克毡房
  • ⚠️ 5月初草原刚返青，草色嫩黄，别期待盛夏景象
  • 下午：天山花海 💐（霍城县芦草沟镇，距伊宁~60km）— 万亩薰衣草基地
  • ⚠️ 花期预警：薰衣草通常6月中旬盛开！5月初只能看到幼苗田
  • 替代：不想看"绿色麦田"可在伊宁多逛（六星街/丝路之光夜市/林则徐纪念馆）
  • 🏨 住宿：伊宁市（连续第二晚）
  • 💰 预估：油费¥75 + 门票¥80(托乎拉苏) + 餐饮¥130 + 住宿¥500 = ~¥785
6 5/6 新源 → 伊宁市（收尾）
188km · ~2.5h
  • 睡个懒觉~ 最后半天悠闲度过
  • 上午：六星街（俄罗斯风情+手风琴博物馆）/ 林则徐纪念馆 / 汉人街买特产
  • 13:00 伊宁机场还车（留足检查时间！）
  • 16:00 伊宁起飞 → 约20:00 成都落地
  • 🏨 无需住宿（当晚回成都）
  • 💰 预估：餐饮¥60 = ~¥60（不含机票和特产购物）

💡 省钱技巧

  • 两人同行：住宿和油费平摊后人均可控在 ¥2,500-3,000（不含机票租车）
  • 伊宁连住两晚：D4-D5都住伊宁不用换酒店，可谈连住折扣
  • 自备干粮：赛里木湖景区内餐饮贵，提前在精河/博乐超市采购
  • 门票优化：赛里木湖自驾环湖公路只买门票¥70即可；区间车非必须
  • 五一预订：越靠近五一酒店越贵甚至满房，现在看到的价格还算合理
  • 租车建议：提前在各大平台比价（神州/一帆/携程），SUV推荐

🚗 自驾关键提醒
  • 精伊牧道：必须SUV/越野车；出发前查「新疆交警」公众号或检查站；非铺装路面约50-80km
  • 加油：G219段加油站间距大，见站加满。精伊牧道无加油站
  • 信号：山区信号差，提前下好离线地图（奥维互动地图/高德离线包）
  • 检查站：新疆检查站频繁，随身带身份证
  • 衣物：早晚0°C~20°C，带羽绒服/冲锋衣；赛里木湖边帽子手套必备
  • 防晒：紫外线极强，防晒霜+墨镜+唇膏不能少
